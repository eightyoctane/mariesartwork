Calendar Update! Several calendar views are available (try selecting them at left, and saving the link for future use). The monthly view is the default, and available from the BMA Menu. Anyone can now add new events to the calendar, monthly view. On the monthly view of the calendar, the calendar "login" (user: anybody pwd: resident), and calendar "control buttons" are at the bottom of the page. Anyone can add events to the calendar. Please do not change the password. It will make the calendar unusable for the other residents.
 

The BMA event calendar is useful if you want to a stake in the ground for a Community Event.

Have you gotten an email event notification and then lost track of it?

If you're sending out email notifications which affect all community members, help everyone out by using the calendar. The Event Calendar has an individual login so that it can be book marked and used directly. This means that you don't have to first log into the BMA website and then also log in to the calendar.

NO NEED TO LOG IN TO VIEW THE CALENDAR! You can view the event calendar without logging in. Logging in is used only for creating an event.

Click on the event to see more detail, like location and notes.

The Event Calendar login area is located at the bottom left corner of the screen:

There is one general login for everyone; user: anybody password: resident. If you have an event which affects a number of BMA residents, the event calendar is a way to make that event visible, and at the same time, insure it doesn't get lost in email.
